Graph#

class Graph(**kwargs)[source]#

Bases: Data

Extends Data class to include additional checks and functionlities.

Initialize the object by setting the node features, edge index, edge attributes, and positions. The edge index is preprocessed to make the graph undirected if required. For more details, see the torch_geometric.data.Data()

Parameters:

  • x (torch.Tensor, LabelTensor) – Optional tensor of node features (N, F) where F is the number of features per node.

  • edge_index (torch.Tensor) – A tensor of shape (2, E) representing the indices of the graph’s edges.

  • pos (torch.Tensor | LabelTensor) – A tensor of shape (N, D) representing the positions of N points in D-dimensional space.

  • edge_attr (torch.Tensor | LabelTensor) – Optional tensor of edge_featured (E, F') where F' is the number of edge features

  • undirected (bool) – Whether to make the graph undirected

  • kwargs (dict) – Additional keyword arguments passed to the Data class constructor.
